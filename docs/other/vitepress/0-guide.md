## 本站安装历程

::: tip

**Node.js 18 及以上版本。**<br>
**通过命令行界面 (CLI) 访问 VitePress 的终端。**<br>
**支持 Markdown 语法的编辑器。**<br>
**推荐 VSCode 及其官方 Vue 扩展。**<br>

:::

- 安装nvm

> [nvm安装教程](/other/nvm/0-guide.html)


- 安装yarn

> ` npm install -g yarn `

- 安装vitepress

> ` yarn add -D vitepress `

- 初始化Vitepress

> ` yarn vitepress init `
